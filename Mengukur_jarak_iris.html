Mengukur Jarak Dataset Iris#

Mengukur jarak merupakan komponen utama dalam algoritma clustering berbasis jarak. Pada Dataset Iris, semua atribut bertipe numerik, sehingga jarak antar data dapat dihitung menggunakan beberapa metode jarak seperti:

  • Euclidean Distance

  • Manhattan Distance

  • Minkowski Distance

1. Euclidean Distance#

Rumus

\[ d(x,y)=\sqrt{\sum_{i=1}^{n}(x_i - y_i)^2} \]

Euclidean Distance digunakan untuk menghitung jarak garis lurus antara dua titik dalam ruang multidimensi.

Contoh Perhitungan D1 dan D2

\[ d(D_1,D_2)=\sqrt{(5.1-4.9)^2+(3.5-3.0)^2+(1.4-1.4)^2+(0.2-0.2)^2} \]
\[ =\sqrt{0.04+0.25+0+0} \]
\[ =\sqrt{0.29} \]
\[ =0.538 \]

#2.Manhattan Distance Rumus

\[ d(x,y)=\sum_{i=1}^{n}\left|x_i - y_i\right| \]

Manhattan Distance menghitung jumlah selisih absolut antar atribut.

Contoh Perhitungan D1 dan D2 $\( d(D_1,D_2)=|5.1-4.9|+|3.5-3.0|+|1.4-1.4|+|0.2-0.2| \)$

\[ =0.2+0.5+0+0 \]
\[ =0.7 \]

#3.Minkowski Distance (p = 3) Rumus

\[ d(x,y)=\left(\sum_{i=1}^{n}|x_i-y_i|^p\right)^{\frac{1}{p}} \]

Dengan:

  • p = parameter jarak

  • p = 1 → Manhattan Distance

  • p = 2 → Euclidean Distance

Minkowski merupakan generalisasi dari Euclidean dan Manhattan.

Contoh Perhitungan D1 dan D2

\[ d(D_1,D_2)=\left(|5.1-4.9|^3+|3.5-3.0|^3+|1.4-1.4|^3+|0.2-0.2|^3\right)^{\frac{1}{3}} \]
\[ =\left(0.008+0.125+0+0\right)^{\frac{1}{3}} \]
\[ =(0.133)^{\frac{1}{3}} \]
\[ =0.510 \]

#Analisis Singkat

1. Diagonal Bernilai Nol

Nilai diagonal pada matriks jarak selalu 0, karena jarak suatu data terhadap dirinya sendiri adalah nol.

2. Sifat Simetris

Matriks jarak bersifat simetris, artinya: $\( d(A,B) = d(B,A) \)$

#3. Kesimpulan Kemiripan

Data 2 dan Data 3 memiliki jarak paling kecil pada semua metode.

Contoh:

  • Euclidean = 0.300

  • Manhattan = 0.5

  • Minkowski = 0.257

Hal ini menunjukkan bahwa Data 2 dan Data 3 merupakan bunga yang paling mirip.

#4. Perbedaan Nilai Metode

  • Manhattan Distance memberikan nilai lebih besar karena menjumlahkan selisih secara linear.

  • Euclidean Distance mempertimbangkan jarak diagonal sehingga nilainya lebih kecil.

  • Minkowski Distance merupakan bentuk umum yang dapat menyesuaikan nilai parameter 𝑝 p.

2. Preprocessing Data

Standardisasi Data (Z-Score/Min-Max): Dilakukan normalisasi agar nilai total_bill (puluhan dollar) dan tip (satuan dollar) memiliki skala yang sama, sehingga model tidak bias.

Partisi Data (80:20):

Karena tidak ada node Partitioning otomatis, kita menggunakan Row Splitter secara manual:

  • 80% Training (195 data) → Untuk melatih model agar mengenali pola perokok.

  • 20% Testing (49 data) → Untuk menguji keakuratan model.

import knime.scripting.io as knio
import pandas as pd
from sklearn.naive_bayes import GaussianNB

# 1. Ambil data dari CSV Reader
df = knio.input_tables[0].to_pandas()

# 2. Tentukan Fitur (X) dan Target (y)
# Kita pakai kolom total_bill, tip, dan size sebagai penentu
X = df[['total_bill', 'tip', 'size']]
y = df['smoker']

# 3. Buat Model Naive Bayes
model = GaussianNB()
model.fit(X, y)

# 4. Buat Prediksi
df['Prediction'] = model.predict(X)

# 5. Kirim kembali ke KNIME
knio.output_tables[0] = knio.Table.from_pandas(df)

a.CSV Reader.#

CSV Reader adalah node yang berfungsi untuk membaca file teks dengan format .csv (Comma Separated Values). File CSV sebenarnya hanyalah teks biasa di mana setiap kolom dipisahkan oleh tanda koma atau titik koma.

b. Python Script#

Script Python di KNIME tersebut digunakan untuk membangun model klasifikasi menggunakan algoritma Naive Bayes. Pertama, library yang dibutuhkan diimpor yaitu knime.scripting.io untuk menghubungkan data dari KNIME ke Python, pandas untuk mengolah data dalam bentuk tabel, dan GaussianNB dari sklearn.naive_bayes sebagai algoritma klasifikasi. Data kemudian diambil dari node sebelumnya (misalnya CSV Reader) dan diubah menjadi DataFrame pandas. Setelah itu ditentukan fitur (X) yaitu kolom total_bill, tip, dan size sebagai variabel input, serta target (y) yaitu kolom smoker yang akan diprediksi. Selanjutnya dibuat model Gaussian Naive Bayes dan dilakukan proses training menggunakan data tersebut. Model yang sudah dilatih kemudian digunakan untuk memprediksi data yang sama, dan hasil prediksi disimpan ke dalam kolom baru bernama Prediction. Terakhir, DataFrame yang sudah berisi hasil prediksi dikirim kembali ke KNIME sebagai output agar bisa digunakan pada node selanjutnya atau ditampilkan.

c.Hasil dan Evaluasi#

Hasil analisis menggunakan model Gaussian Naive Bayes menunjukkan bahwa model berhasil memprediksi status pelanggan (perokok atau bukan) berdasarkan variabel total_bill, tip, dan size. Evaluasi yang dilakukan melalui node Scorer menunjukkan tingkat akurasi model dalam mencocokkan data aktual dengan hasil prediksi, yang divisualisasikan melalui Confusion Matrix. Secara keseluruhan, model ini terbukti efektif dalam mengenali pola probabilitas dari dataset tips.csv, meskipun terdapat beberapa kesalahan prediksi yang wajar terjadi karena adanya kemiripan pola konsumsi antara pelanggan perokok dan non-perokok.

Berdasarkan analisis menggunakan algoritma Gaussian Naive Bayes, model berhasil memprediksi status pelanggan perokok atau bukan dengan memproses variabel total_bill, tip, dan size. Evaluasi menggunakan node Scorer menunjukkan performa model melalui nilai akurasi yang diperoleh dari perbandingan antara data asli dan hasil prediksi pada Confusion Matrix. Secara keseluruhan, model mampu mengenali pola probabilitas dari dataset secara otomatis, meskipun terdapat selisih kecil (error) yang disebabkan oleh kemiripan karakteristik perilaku antara kelompok perokok dan non-perokok dalam data tersebut.

Berdasarkan dataset tips.csv, model Gaussian Naive Bayes berhasil memprediksi apakah seorang pelanggan adalah perokok atau bukan dengan memanfaatkan variabel total_bill, tip, dan size. Evaluasi menggunakan node Scorer menunjukkan bahwa model memiliki kemampuan klasifikasi yang cukup baik, terlihat dari perbandingan data asli dan hasil prediksi pada tabel Confusion Matrix. Hasil ini membuktikan bahwa pola pengeluaran pelanggan dapat digunakan sebagai dasar prediksi perilaku melalui pendekatan machine learning berbasis Python di KNIME.

3. Kesimpulan Evaluasi Mode#

  • Total Data yang Dianalisis: Sebanyak 244 baris data dari dataset tips.csv berhasil diproses oleh model.

  • Variabel Prediktor (Fitur): Menggunakan 3 variabel angka (total_bill, tip, dan size) sebagai dasar perhitungan probabilitas Naive Bayes.

  • Hasil Klasifikasi: Model membagi data ke dalam 2 kategori target, yaitu “Yes” (Perokok) dan “No” (Bukan Perokok).

  • Evaluasi Akurasi: Melalui node Scorer, model menunjukkan performa prediksi yang diukur dalam persentase (Contoh: jika akurasi 0.7 berarti 70% data berhasil ditebak dengan benar).

  • Distribusi Data: Model Gaussian Naive Bayes bekerja optimal karena mampu menghitung nilai rata-rata (mean) dan varians dari data transaksi yang bervariasi antara \(3.07 hingga \)50.81.

Poin-poin ini menunjukkan bahwa analisis kamu tidak hanya sekadar “jalan”, tapi didukung oleh jumlah data yang nyata.
